# Prerequisites: 

### Ensure pip installer is available:
``` 
python3 -m ensurepip --upgrade
```
### Install required packages:
```
pip3 install opencv_contrib_python
```

```
pip3 install numpy
```    
### Run the program:
```
python3 hough_transform.py
```
